
## 核心改进

### 1. 专业科学配色方案
```
Tube 1 (阳性): #2196F3 - 蓝色 (清晰、专业)
Tube 2 (阳性): #9C27B0 - 紫色 (优雅、醒目)
Tube 3 (阳性): #FF9800 - 橙色 (温暖、活力)
Tube 4 (对照): #757575 - 灰色 (低调、区分)
```

### 2. 视觉层次优化
- **阳性组**：
  - 实线，线宽 3.5px
  - 渐变填充区域（25% → 5% → 0%）
  - 柔和阴影效果
  - 数据点 4.5px

- **对照组**：
  - 虚线（8px-4px 间隔）
  - 无填充区域
  - 轻微阴影
  - 数据点 4px

### 3. 曲线平滑度
- curveSmoothness: 0.4 (更自然的曲线)
- preventCurveOverShooting: true (防止过冲)
- 更流畅的动画：1500ms

### 4. 字体和标签
- 所有标签：12px
- 字母间距：0.5
- 透明度：80%
- 双语标签：中文 + 英文

### 5. 网格和边框
- 仅水平网格线（透明度 15%）
- 边框加粗到 2px
- 极简风格

### 6. 交互体验
- Tooltip：黑色半透明背景
- 触摸点：6px 大小
- 虚线指示器
- 精确到小数点后 2 位

### 7. 图表尺寸
- 分析页面：280px
- 统计页面：300px
- 更多留白空间

## 特色功能

1. **渐变填充**：阳性组曲线下方有优雅的渐变填充
2. **阴影效果**：曲线带有柔和阴影，增强立体感
3. **防过冲**：曲线不会出现不自然的波动
4. **智能刻度**：自动计算最优的 Y 轴间隔
5. **空状态**：无数据时显示友好提示
