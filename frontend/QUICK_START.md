# 🚀 快速启动指南

## 第一步：安装依赖

打开终端，进入项目目录：

```bash
cd d:\fluo_app\frontend
flutter pub get
```

## 第二步：启动后端服务

在另一个终端窗口：

```bash
cd d:\fluo_app\gemini_backen
python main.py
```

确保后端运行在 `http://127.0.0.1:8000`

## 第三步：配置 ADB（仅 Android）

如果使用 Android 设备测试：

```bash
# 连接设备后，执行端口转发
adb reverse tcp:8000 tcp:8000
```

## 第四步：运行应用

```bash
cd d:\fluo_app\frontend
flutter run
```

或者在 VS Code / Android Studio 中直接点击运行按钮。

## 🎯 测试流程

1. **检查连接状态**

## ⚠️ 常见问题

### 问题 1：后端连接失败

**症状**：右上角显示红色"未连接"

**解决方案**：
1. 确认后端服务已启动
2. 检查 ADB 端口转发：`adb reverse tcp:8000 tcp:8000`
3. 重启应用

### 问题 2：图片上传失败

**症状**：点击"开始分析"后提示上传失败

**解决方案**：
1. 检查网络连接
2. 查看后端日志
3. 确认图片格式正确（JPG/PNG）

### 问题 3：依赖安装失败

**症状**：`flutter pub get` 报错

**解决方案**：
```bash
flutter clean
flutter pub get
```
